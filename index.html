AriCalculator is a home made pocket calculator.

Hardware

The hardware is based on Freescale's MC9S12G micro controller family. It has a 38 button key pad and a ST7565R based 128×64 dot matrix LCD display (optionally back lit). As of RevC, the AriCalculator also has a general purpose 3.3V SPI expansion port.

Software

The software for the AriCalculator will be written in Forth. This will make it easier to port it to other platform and to add new features.
