AriCalculator is a project I plan to do with my daughter. The plan is to build our own calculator, by the time she is allowed to use one in school. Let’s see how we’ll do.

Hardware

The current hardware is based on the S12G Micro EVB using a MC9S12G128 micro controller from Freescale. However, this board can be easily replaced by other microcontroller boards of this form factor. The micro controller board is mounted on a milled main board, which also holds a 5×7 key matrix and a 128×64 dot matrix reflective LCD (ST7565R connected via SPI).

Software

The software for the AriCalculator will be written in Forth. Forth has the advantage, that…

  • …the firmware will be platform independant and can be run on other MCUs with compatible Forth implementation.
  • …it is a great language to teach programming.
  • …compiler and IDE are running on the MCU itself. All we need on the PC side is a terminal emulator and an editor.
